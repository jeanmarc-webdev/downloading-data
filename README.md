DOWNLOADING DATA PROJECT — PYTHON CODE

This repository contains the Python scripts for the Downloading Data project, based on Python Crash Course by Eric Matthes.
The project focuses on data visualization and working with CSV and GeoJSON data.

PROJECT STRUCTURE

downloading-data/

.venv/ → Python virtual environment
download_data.py → Main script for downloading CSV/GeoJSON files
temperature_chart.py → Plotting daily temperature data
earthquake_map.py → Plotting earthquake data on world maps
utils.py → Helper functions (if any)
screenshots/ → Images generated by the scripts

DEPENDENCIES

The project uses Python 3.x and several libraries, including:

matplotlib
pandas
plotly
plus other third-party libraries installed locally

Optional:
You can generate a requirements file using:
pip freeze > requirements.txt

To install dependencies:
pip install -r requirements.txt

HOW TO RUN

Activate virtual environment:

Windows:
.venv\Scripts\activate

Mac/Linux:
source .venv/bin/activate

Run scripts:

python download_data.py
python temperature_chart.py
python earthquake_map.py

NOTES

.venv folder is ignored using .gitignore
Scripts generate visual outputs and charts
Project is for learning Python data visualization
